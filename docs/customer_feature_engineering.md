# Customer Feature Engineering

## Features Created

### 1. age_group
Groups customers based on age.

- Teen
- Young Adult
- Adult
- Middle Age
- Senior
- Elder

### 2. is_active_member
Indicates whether the customer has a valid membership.

- 1 = Active Member
- 0 = Unknown Membership

### 3. receives_fashion_news
Indicates whether the customer receives fashion news.

- 1 = Yes
- 0 = No

### 4. is_active_customer
Customer activity indicator.

- 1 = Active
- 0 = Inactive
